# 🥁 DrumXRoll – XR Drum Learning System  
*A Unity-based multimodal drumming tutor for Meta Quest 3*

---

## 📌 Overview  
**DrumXRoll** is an extended-reality (XR) drum-learning system built in **Unity** for the **Meta Quest 3**.  
It teaches drumming fundamentals through a **piano-roll–style visualizer**, **interactive lesson modes**, and **physical drum-set integration**.  
The system blends gesture tracking, spatial cues, and real-time feedback to help learners practice rhythm, timing, and limb coordination.

Key features include:

- 🎼 Piano-roll note visualizer  
- 🥁 Drum-hit detection & interaction logic  
- 🎧 Metronome, timing windows, audio feedback  
- 🕹️ Multiple lesson modes (Watch & Listen, Try Yourself, Compose on the Fly, Constraint & Create, etc.)  
- 🎮 XR integration for Meta Quest 3  
- 🧩 UI/UX flows, managers, scene controllers, and core scripts  

This repository contains the **Unity project scripts**, **project settings**, and **package definitions** required for both linear and spatial versions to run DrumXRoll.

## 🚀 How to Open the Project

1. Clone or download this repository.
2. Download the **Assets folder** from the Google Drive link.
3. Place the downloaded `Assets/` folder in the root of the project so the structure becomes:
/Assets/
/Packages/
/ProjectSettings/
/UserSettings/ (optional)
4. Open the project in **Unity 2022.3.55f1** (use your project's exact version).
5. Wait for Unity to reimport assets on first load.

## 🥁 About DrumXRoll  
DrumXRoll is designed to make drumming practice more immersive and intuitive by combining:

- **Visual learning** via spatialized falling notes  
- **Kinesthetic learning** using real and virtual drums  
- **Adaptive lesson flows** suited to beginner and intermediate drummers  
- **Instant feedback** for timing accuracy and consistency  

The system is developed under the **Human-X Interactions Lab (HXIL)** at **De La Salle University** for research on XR-based music learning.
